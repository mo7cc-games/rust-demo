# rust-demo

rust 的学习 Demo

## 常用命令

```bash

## 安装依赖
cargo install cargo-watch
cargo install cargo-edit

## 运行
cargo run

## 编译调试
cargo build

## 编译发布
cargo build --release

# 当代码文件变化时，自动重新运行程序。
cargo watch -x run

# 当代码变化时，自动运行测试。
cargo watch -x test

# 文件变化时自动构建 release 版本。
cargo watch -x "build --release"

```
